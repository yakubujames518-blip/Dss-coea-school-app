# Backend API Structure

## Directory Structure

```
/backend
    ├── controllers
    ├── middleware
    ├── models
    ├── routes
    ├── services
    └── utils
```

## Files

- **controllers/**: Contains all the controllers for handling requests.
- **middleware/**: Contains all middleware functions.
- **models/**: Contains database models.
- **routes/**: Contains all the routes for the API.
- **services/**: Contains business logic layers.
- **utils/**: Contains utility functions.
